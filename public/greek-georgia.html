Georgia + Greek Characters

Georgia has full Greek glyph support built in. Unlike Boska, the Greek letters will match the Latin ones perfectly — same stroke weight, same contrast, same design.

Original — All Latin
estiasi
Current wordmark. All Latin characters in Georgia.
e-s-t-i-a-s-i
Option A — Greek ε for e
εstiasi
Opening epsilon. Signals Greek origin immediately. Georgia renders it natively — same weight, same style.
ε-s-t-i-a-s-i
Option B — Greek ι for both i's (dotless)
estιasι
Both i's replaced with iota — no dot above. Cleaner, more minimal. Subtle uniqueness.
e-s-t-ι-a-s-ι
Option C — ε + ι (epsilon + iotas)
εstιasι
Three Greek characters. Most distinctive while still readable. Clearly signals Greek origin.
ε-s-t-ι-a-s-ι
Option D — ε + τ + ι (epsilon + tau + iotas)
εsτιasι
Four Greek characters. Tau (τ) has a slightly different crossbar from Latin t in Georgia.
ε-s-τ-ι-a-s-ι
Option E — Just dotless ι for final i
estiasι
Most subtle option. Only the last letter changed. The missing dot on the final i is a quiet signature.
